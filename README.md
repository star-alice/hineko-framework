## 🌺 Hineko Framework
> This is a simple framework made for luau based on the well-known `Flamework` for roblox-ts for Roblox Luau. It is being made for study purposes, you can report errors and improvements if needed!

## 🔨 How it works?
> You can check the folder `features` inside `src/server`, which contains two test services that has all the actual implementations and tests from Hineko Framework. Also, `src/client` for some client-side examples!
